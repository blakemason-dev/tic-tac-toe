# tic-tac-toe

## About
Play a simple game of tic-tac-toe with friends online

## Tech Stack
- Phaser
- Colyseus
- Typescript
- React
- Express

## How to Run
Clone, install and run the demo with the following:

    git clone git@github.com:blakemason-dev/tic-tac-toe.git
    cd tic-tac-toe/server
    npm install
    npm start
    cd tic-tac-toe/client
    npm install
    npm run dev
